# Article Normalizer

## Описание

Инструмент для нормализации научных статей в формате .docx.

Решает практическую задачу: есть набор документов от разных авторов с разной структурой и форматированием — необходимо привести их к единому виду без ручной обработки каждого файла.

На входе — неструктурированные или частично структурированные документы.  
На выходе — структурированные данные, готовые к сборке каталога.

---

## Проблема

Реальные статьи часто не соответствуют формату:

- отсутствуют явные заголовки ("Аннотация", "Abstract")
- перемешаны русский и английский блоки
- авторы записаны в разных форматах
- встречаются дубликаты и неполные версии
- нестабильное форматирование (пробелы, переносы, мусор)

Ручная обработка таких данных плохо масштабируется.

---

## Решение

Реализован ETL-пайплайн:

- разбор `.docx` через `python-docx`
- сегментация текста по маркерам
- извлечение ключевых блоков:
  - УДК
  - название (RU / EN)
  - авторы
  - аннотация
  - ключевые слова
  - основной текст
  - список источников
- очистка текста
- fallback-логика при нарушенной структуре
- классификация документов:
  - ok
  - partial
  - invalid
  - ignored
